## 系统介绍

### 平台概述
热力方舟是基于区块链技术的P2P热能交易平台，旨在实现热能资源的高效分配和去中心化交易。平台连接热能生产者和消费者，提供安全、透明的交易环境。

### 主要功能
- **用户管理**: 注册、登录、身份认证
- **热能交易**: 买卖热能、订单撮合、交易结算
- **支付系统**: 多种支付方式、资金管理
- **区块链集成**: 智能合约、交易记录、安全保障
- **数据监控**: 实时能源生产数据、交易历史

### 适用用户
- **热能生产者**: 拥有热能生产设备的用户
- **热能消费者**: 需要购买热能的用户
- **能源交易商**: 专业从事能源交易的用户

## 用户注册和认证

### 注册流程
1. **访问平台**: 打开热力方舟网站 (http://localhost:3000)
2. **点击注册**: 在首页点击"注册"按钮
3. **填写信息**:
   - 用户名 (3-20位字母数字)
   - 邮箱地址 (用于验证和找回密码)
   - 密码 (8位以上，包含大小写字母和数字)
   - 确认密码
4. **同意协议**: 阅读并同意用户协议和隐私政策
5. **完成注册**: 点击"注册"按钮，系统将发送验证邮件

### 邮箱验证
1. **查收邮件**: 登录注册邮箱，查收热力方舟发送的验证邮件
2. **点击链接**: 点击邮件中的验证链接
3. **验证成功**: 页面跳转至登录界面，邮箱验证完成

### 登录流程
1. **输入凭证**: 在登录页面输入用户名/邮箱和密码
2. **安全验证**: 完成人机验证（如需要）
3. **登录成功**: 系统验证通过后进入用户仪表盘

### 密码管理
- **修改密码**: 在个人设置中修改密码
- **忘记密码**: 点击"忘记密码"，通过邮箱重置
- **密码安全**: 定期更换密码，避免使用简单密码

## 交易操作指南

### 市场概览
在交易页面可以查看：
- **实时价格**: 当前热能交易价格走势
- **深度图**: 买卖订单深度分布
- **交易对**: 支持的热能交易品种
- **成交量**: 24小时交易量统计

### 买入热能
1. **选择交易对**: 选择要交易的热能品种
2. **输入数量**: 输入要购买的热能数量（kWh）
3. **设置价格**: 选择市价单或限价单
   - **市价单**: 按当前最优价格立即成交
   - **限价单**: 设置期望的买入价格
4. **确认订单**: 查看订单详情，确认无误
5. **提交订单**: 点击"买入"提交订单

### 卖出热能
1. **选择交易对**: 选择要交易的热能品种
2. **输入数量**: 输入要出售的热能数量（kWh）
3. **设置价格**: 选择市价单或限价单
   - **市价单**: 按当前最优价格立即成交
   - **限价单**: 设置期望的卖出价格
4. **确认订单**: 查看订单详情，确认无误
5. **提交订单**: 点击"卖出"提交订单

### 订单管理
- **查看订单**: 在"我的订单"页面查看所有订单
- **订单状态**: 待成交、部分成交、已成交、已取消
- **撤销订单**: 未成交的订单可以随时撤销
- **订单历史**: 查看已完成订单的详细信息

### 交易撮合
平台采用先进先出（FIFO）撮合机制：
1. **价格优先**: 相同价格下按时间顺序撮合
2. **自动匹配**: 系统自动匹配买卖订单
3. **即时结算**: 成交后立即进行资金和热能结算

## 钱包管理

### 资金充值
1. **进入钱包**: 点击导航栏"钱包"进入资金管理页面
2. **选择充值**: 点击"充值"按钮
3. **选择方式**: 支持微信支付、支付宝、银行转账
4. **输入金额**: 输入充值金额（最低10元）
5. **完成支付**: 按照支付平台指引完成支付
6. **到账确认**: 支付成功后资金实时到账

### 资金提现
1. **进入钱包**: 点击导航栏"钱包"进入资金管理页面
2. **选择提现**: 点击"提现"按钮
3. **选择方式**: 支持银行卡提现
4. **输入金额**: 输入提现金额（最低50元）
5. **确认提现**: 输入支付密码确认提现
6. **处理时间**: 1-3个工作日内到账

### 交易记录
- **充值记录**: 查看所有充值操作和状态
- **提现记录**: 查看提现申请和处理进度
- **交易记录**: 查看热能交易的资金流水
- **导出记录**: 支持导出CSV格式的交易记录

### 资金安全
- **双重验证**: 重要操作需要短信或邮箱验证
- **交易限额**: 设置单日和单笔交易限额
- **资金冻结**: 交易中的资金会被临时冻结
- **风险提示**: 大额交易会有风险提示

## 能源生产监控

### 设备连接
1. **添加设备**: 在"设备管理"页面添加热能生产设备
2. **设备信息**: 填写设备型号、功率、位置等信息
3. **连接配置**: 配置设备通信参数和协议
4. **在线状态**: 查看设备连接状态和数据传输情况

### 数据监控
- **实时数据**: 查看设备实时生产数据
- **生产统计**: 日、周、月生产量统计
- **效率分析**: 设备运行效率分析报告
- **异常预警**: 设备异常状态自动预警

### 生产记录
- **自动记录**: 系统自动记录能源生产数据
- **区块链存证**: 重要生产数据上链存储
- **数据导出**: 支持生产数据导出和分析
- **报表生成**: 自动生成生产报表和收益分析

### 收益管理
- **收益计算**: 根据生产数据自动计算收益
- **收益分配**: 设置收益分配比例和规则
- **提现申请**: 收益达到阈值后可申请提现
- **税务报告**: 生成税务相关的收益报告

## 常见问题解答

### 账户相关

**Q: 忘记密码怎么办？**
A: 点击登录页面的"忘记密码"，输入注册邮箱，系统将发送密码重置链接。

**Q: 如何修改个人信息？**
A: 登录后进入"个人设置"页面，可以修改昵称、头像、联系方式等信息。

**Q: 账户被锁定怎么办？**
A: 连续输错密码会导致账户锁定，30分钟后自动解锁，或联系客服手动解锁。

### 交易相关

**Q: 订单为什么没有立即成交？**
A: 限价单需要等待匹配的对手单，市价单会按当前最优价格立即成交。

**Q: 交易手续费是多少？**
A: 目前平台交易手续费为0.1%，具体费率以平台公告为准。

**Q: 如何撤销未成交的订单？**
A: 在"我的订单"页面找到待成交订单，点击"撤销"按钮即可。

### 支付相关

**Q: 充值多久到账？**
A: 微信支付和支付宝实时到账，银行转账1-3个工作日到账。

**Q: 提现需要手续费吗？**
A: 单笔提现手续费为2元，具体以银行实际收费为准。

**Q: 为什么提现被拒绝？**
A: 可能原因：账户未实名认证、提现金额超过限额、安全风险等。

### 区块链相关

**Q: 区块链交易确认需要多久？**
A: 通常在几分钟内完成确认，具体时间取决于网络拥堵情况。

**Q: 交易记录在哪里查看？**
A: 在"区块链浏览器"中可以查看所有上链的交易记录。

**Q: 智能合约安全吗？**
A: 所有智能合约都经过专业安全审计，确保交易安全可靠。

- 用户调研：参与平台定期组织的用户调研

### 安全提示

1. **账户安全**: 不要与他人共享账户信息，定期修改密码
2. **资金安全**: 大额资金建议分散存储，启用双重验证
3. **交易安全**: 确认交易信息后再提交，避免误操作
4. **设备安全**: 定期检查生产设备，确保数据安全传输
5. **防范诈骗**: 警惕虚假客服和钓鱼网站，只使用官方渠道
